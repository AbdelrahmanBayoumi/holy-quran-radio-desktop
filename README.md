## Installation

### Prerequisites

- [Node.js](https://nodejs.org/) (v14.x or later)
- [npm](https://www.npmjs.com/) (v6.x or later)

### Steps

1. Clone the repo:

   ```bash
   git clone https://github.com/AbdelrahmanBayoumi/holy-quran-radio-desktop.git
   cd holy-quran-radio-desktop
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Run the app:

   ```bash
   npm start
   ```

## Packaging
